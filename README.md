### Тесты

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AntiSlang/KR_1_MELNIK.git
   cd KR_1_MELNIK
   ```

2. Восстановите зависимости:
   ```bash
   dotnet restore
   ```

3. Запустите тесты:
   ```bash
   cd .\KR_1_MELNIK.Tests\
   dotnet test
   ```

### Структура проекта

- KR_1_MELNIK — основной проект
- KR_1_MELNIK.Tests — проект с модульными тестами
